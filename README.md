# codesaga-day28 - Number Guessing Game Variations

Mini Project 3 of Codesaga Journey

## Files Included

### 1. basic_guess_game.py
Basic guessing game where computer selects a random number and user guesses.

### 2. limited_attempts_game.py
User gets only 5 chances to guess the correct number.

### 3. very_close_hint_game.py
If the guessed number is very close (±2), a special hint is shown.

### 4. reverse_guess_game.py
User thinks of a number and computer guesses using low-high logic.

### 5. repeat_guess_check_game.py
If the user enters the same number again, the game warns using list checking.
